Publish NPM package

To publish npm package follow below steps:
1. need to login to npm by using below cmd
npm login

2. To push your package to npm, run below cmd
npm publish <package_name>

3. To remove the package from npm
npm unpublish <package_name> --force

4. After pubished the package to npm first time, again want to add more changes then change version attribute value in package.json
